# AI Compliance Workspace

A practical browser-based workspace for AI governance, risk tracking, evidence collection, and audit preparation.

## Purpose

AI Compliance Workspace is a portfolio project focused on structured AI governance work.

The goal is to help track:

- AI systems
- owners and business purpose
- risk classification
- AI-specific risks
- controls
- evidence
- incidents and issues
- governance decisions
- open gaps
- review dates
- audit preparation reports

## Important Disclaimer

This project is not a legal compliance engine.

It does not provide legal advice, certify compliance, or guarantee compliance with the EU AI Act, ISO/IEC 42001, GDPR, NIS2, SOC 2, or any other framework.

It is a practical workspace for organizing governance, risk, evidence, and audit-readiness information.

Do not enter real confidential, personal, customer, regulated, or sensitive company data into the public demo.

## Planned MVP Features

- Dashboard
- AI System Inventory
- Risk Classification Helper
- AI Risk Register
- Controls & Evidence
- Decision Journal
- Incident / Issue Log
- Framework Mapping
- Markdown report export
- JSON import/export
- Local browser storage only
